# PneumoniaClassification
>  X-ray 이미지셋을 통해 Pneumonia 병변을 분류하는 모델입니다.
>
> Keras를 통한 CNN 구현으로 이미지 분류 모델을 개발하였습니다.
>
> 흉부 X-ray 이미지 셋 출처 :

> https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia
